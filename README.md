# Stromkosten-Rechner

Ein einfacher Stromkosten-Rechner als Web-App.

## GitHub Pages

Die App ist fuer GitHub Pages konfiguriert:

1. Gehe zu Settings > Pages in deinem Repository
2. Waehle als Source den Branch main aus
3. Waehle als Ordner / (root) aus
4. Klicke auf Save

Deine App wird dann unter https://seeymore2.github.io/test/ verfuegbar sein.

## Lokale Nutzung

Einfach die index.html Datei in einem Browser oeffnen.
